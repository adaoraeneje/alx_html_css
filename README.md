# SmileSchool – Case Study

> A pixel-perfect recreation of a marketing website using pure HTML5 and CSS3 — no JavaScript, no frameworks, just code.

---

## Overview

This project is a responsive, fully semantic webpage built to reflect the [SmileSchool](https://www.figma.com) marketing website design as shared by ALX. It is part of the **CSS Advanced Project** in the ALX Software Engineering program.

The core objective was to take a previously built HTML skeleton and apply advanced CSS styling to achieve layout accuracy, visual balance, and responsiveness — without the help of JavaScript or external libraries.

---

## Goals

- Bring a static HTML structure to life using only **pure CSS**
- Recreate the design layout with **pixel-level accuracy** based on Figma
- Demonstrate **responsiveness**, **typography hierarchy**, and **visual clarity**
- Understand and apply **semantic HTML5** practices for accessibility and structure

## Key Features

### Semantic HTML Structure
- All elements are properly nested inside `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` for better accessibility and SEO.

### Fully Responsive Layout
- The site adjusts across screen sizes using **flexbox** and **media queries**.
- Designed mobile-first and scales to large desktops.

### Background Imagery and Positioning
- Hero sections use full-width background images with **overlaid content**, controlled using positioning and z-index principles.

###  Typography & Spacing
- Proper font weights, font sizes, and spacing are used to reflect visual hierarchy.
- Button states and hover effects enhance interactivity without JS.
